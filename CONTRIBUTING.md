# Contributing

Danke fürs Mitmachen!

## Bitte keine sensiblen Daten
Nicht posten/committen:
- Tokens / API Keys / Authorization Header
- interne URLs/IPs/Hostnames
- secrets.yaml oder vergleichbare Dateien

## Wie du helfen kannst
- Issues kommentieren (Edge-Cases, Verbesserungsvorschläge)
- Pull Requests: kleine, nachvollziehbare Änderungen bevorzugt
- Bitte kurz beschreiben: Was ändert sich? Warum?

## Ziel
Stabil, nachvollziehbar, wenig Flattern („kein Ping-Pong“).
